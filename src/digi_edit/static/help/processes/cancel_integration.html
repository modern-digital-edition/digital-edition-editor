Cancel Task integrationÂ¶

If you wish to cancel a Task integration, click on the cancel integration icon (). This will cancel the integration on the backend, and after a few seconds the icon will update to indicate that there is no integration requested (). After making any additional changes you wish to make, you can then request the integration again.
